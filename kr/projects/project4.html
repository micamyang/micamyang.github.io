네트워크 망 구축 프로젝트

2025.03.10 ~ 2025.03.25

요약

이 프로젝트는 Cisco Packet Tracer 가상 환경을 활용하여 국내외 여러 지사(미국, 부산, 베를린, 제주)와 데이터센터를 포함하는 통합 네트워크 인프라를 설계하고 시뮬레이션하는 것을 목표로 진행되었습니다. 주요 목표는 물리적인 장비 없이도 안정적인 네트워크 연결과 지사 간, 데이터센터와의 트래픽 보안을 강화하는 효율적이고 안전한 가상 통신 환경을 구축하는 것입니다.

역할

저는 베를린 지사 네트워크 구축 및 라우팅 설정 담당으로서, Cisco Packet Tracer 가상 환경 내에서 독일 베를린 지사의 고유한 네트워크 환경을 고려하여 네트워크를 설계하고 설정했습니다. 팀원들과의 적극적인 협업과 소통을 통해 설계한 네트워크가 가상 시스템 내에서 원활히 작동하도록 기여했으며, 시뮬레이션을 통한 문제 해결 과정을 통해 복잡한 네트워크 구조에 대한 깊은 이해와 실제 환경 적용 가능성을 증명할 수 있었습니다. 특히, 가상 시뮬레이터를 통해 네트워크의 라우팅 개념을 체계적으로 이해하고 실습하는 기회가 되었습니다.

결과

Cisco Packet Tracer 시뮬레이션 환경 내에서 각 지사와 본사, 데이터센터 간 네트워크를 성공적으로 연결하고, Site-to-Site VPN 및 Static 라우팅 설정을 통해 안정적이고 보안이 강화된 가상 통신 환경을 구현했습니다. 이는 물리적인 장비 없이도 복잡한 네트워크 구성을 테스트하고 문제 해결 능력을 효과적으로 보여줄 수 있는 중요한 경험이었습니다.

기술

Cisco Packet Tracer, Site-to-Site VPN, Static 라우팅, VLAN, DHCP
