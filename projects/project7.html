Hometown Introduction Website

16.10.2024 ~ 11.12.2024

Overview

This project involved developing a website to introduce my hometown, featuring content on its history, landmarks and local cuisine. A key focus of the project was implementing form-handling functionality using PHP to process user input and enable dynamic interaction. This provided valuable experience in building a website that goes beyond delivering static information by encouraging user engagement and showcasing the ability to develop interactive web applications.
